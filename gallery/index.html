Wall Detection & FoundryVTT Export

AI-powered wall detection extracts walls from battle maps and exports them in FoundryVTT-ready JSON format with coordinates and blocking settings.

Process

  1. Convert battle map to grayscale PNG
  2. Use Gemini 2.5 Flash Image with prompt: "Draw red lines for walls in this battle map. Draw straight lines only. Avoid stairs. Do not outline the frame."
  3. Extract vector polylines from red-lined image using edge detection (morphological closing, contour detection, Douglas-Peucker)
  4. Convert coordinates to FoundryVTT wall format with blocking settings
  5. Generate overlay visualization for manual verification

Scene Artwork Generation

AI-powered scene extraction identifies physical locations from module text and generates atmospheric artwork using Imagen 4.0 Fast.

Process

  1. Extract chapter environmental context using Gemini (environment type, lighting, terrain)
  2. Identify physical location scenes in XML, filtering out NPCs and monsters
  3. For each scene, construct prompt with location description, context, and style constraints
  4. Generate artwork using Imagen 4.0 Fast (parallel processing, 5 concurrent workers)
  5. Save images and metadata with section paths for automatic positioning

Prompt Template

{scene.description}

Location Type: {location_type}
Lighting: {lighting}
Terrain: {terrain}
Atmosphere: {atmosphere}

Style: fantasy illustration, D&D 5e art style, detailed environment, high quality

IMPORTANT CONSTRAINTS:
- Do NOT include any text, words, letters, signs, or writing in the image
- Do NOT depict specific named creatures or characters
- Focus on the physical environment and location details only
- If location_type is "underground", ensure the scene clearly shows it is inside a cave/dungeon
